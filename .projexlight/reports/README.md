# Git Hook Reports

This folder contains reports generated by ProjexLight git hooks.

## Folder Structure

```
reports/
├── pre-commit/     # Immediate fixes required (blocks commit)
│   ├── violations_*.json   # Timestamped violation reports
│   └── latest.json         # Most recent report
├── pre-push/       # Pre-push violations (API failures, conflicts)
│   ├── api_conflicts_*.json   # API duplicate conflicts
│   ├── api_failures_*.json    # API test failures
│   ├── conflicts_latest.json  # Latest conflict report
│   └── failures_latest.json   # Latest failure report
└── todos/          # End-of-sprint cleanup (non-blocking)
    ├── todos_*.json           # From pre-commit
    ├── pre-push_todos_*.json  # From pre-push
    ├── latest.json            # Latest pre-commit TODOs
    └── pre-push_latest.json   # Latest pre-push TODOs
```

## Report Priority

| Folder | Priority | When to Fix |
|--------|----------|-------------|
| `pre-commit/` | **Immediate** | Before committing - blocks commit |
| `pre-push/` | **Immediate** | Before pushing - blocks push |
| `todos/` | End of Sprint | Can be addressed later |

## Using Reports with LLM

Each report contains an `llm_prompt` field with instructions for your AI coding assistant.

**To fix violations:**
```
"Read .projexlight/reports/pre-commit/latest.json and fix the issues"
```

**To fix TODOs:**
```
"Read .projexlight/reports/todos/latest.json and fix the TODO items"
```

**To fix API failures:**
```
"Read .projexlight/reports/pre-push/failures_latest.json and fix the failed tests"
```

## Report Contents

Each JSON report includes:
- `reportType`: Type of report (violations, todos, api_failures, api_conflicts)
- `timestamp`: When the report was generated
- `priority`: "immediate" or "end-of-sprint"
- `summary`: Quick overview of issues
- `items`: Detailed list of issues
- `llm_prompt`: Ready-to-use prompt for AI assistant
- `actionable_commands`: Useful commands to investigate

## Cleaning Up Reports

Reports are automatically timestamped. You can safely delete old reports:

```bash
# Keep only latest reports
rm -f .projexlight/reports/pre-commit/violations_*.json
rm -f .projexlight/reports/todos/todos_*.json
```
